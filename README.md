## サービス概要
愛犬の年齢、体型、運動量、アレルギーなどの情報に基づいて、その子に最適な手作りレシピを提案するサービスです。「何をあげていいかわからない」という飼い主の悩みを解決し、愛犬の健康寿命を延ばすことを目指します。
## このサービスへの思い・作りたい理由
元家庭犬訓練士を職としており、飼い主様からの愛犬の食事への相談が常に絶えませんでした。
「自分の子にあった食事を作りたいが何を使うべきかわからない」
「逆に何を食べさせてはいけないのかわからずハードルが高い」
このような悩みを持つ飼い主様が、気軽に情報を調べられるようにすることで、愛犬の手作りごはんへのハードルも下がり、健康寿命が延びることを期待し、このアプリの開発を決意しました。

## ユーザー層について
主なターゲットは「愛犬の健康を気にしているが、手作りごはんに不安を感じている飼い主」です。
特に下記のような方を想定しています。
・犬を飼ってる方
・愛犬の健康に気を遣っている方
・愛犬が食べてくれるご飯が無く困っている方
・愛犬に手作りご飯を作りたいけど作り方がわからない方

## サービスの利用イメージ
1.愛犬の情報を選択
・ 犬種(小型 / 中型 / 大型)
・ 年齢（仔犬 / 成犬 / シニア）
・ 体型（細身 / 中肉 / 肥満）
・ 運動量（少ない / 普通 / 多い）
・ アレルギー情報

2.レシピの自動提案
・ 登録した情報に基づいて、最適なレシピを 10件提案（一度作ると3日ほど持つため、１０個のレシピで　　一ヶ月持つ計算としている）
・ 食材選択ではなく、アプリ側が愛犬の状態に合わせて厳選

3.レシピの確認・お気に入り登録
・  材料、作り方、栄養情報を確認
・　気に入ったレシピはお気に入りに保存
・　SNS でシェアして他の飼い主にも共有

食材から選ぶのでは無く、健康状態等から厳選された少数から選ぶことができます。
ドックフードだけに頼るのでは無く、自分で愛犬の健康を守りたいというオーナーの気持ちを尊重、また愛犬との絆を深めることができます。

## ユーザーの獲得について
・Xにてハッシュタグを利用し実際の利用方法を投稿。（＃愛犬手作りごはん/#犬のいる暮らし）など
・Instagramで愛犬の写真と共にアプリ紹介
・犬フェスなどオンラインで集まるコミュニティで紹介
・アプリ内でレシピをSNSにシェアできる機能の実装

## サービスの差別化ポイント・推しポイント
犬の手作りごはんの検索アプリはありますが、その子の状態に合わせたごはんの提示をするアプリはないため、差別化が図られます。
「どれにしようかな」と多数のレシピの中から迷わせるのでは無く、少数に絞ることで取捨選択のストレスを無くし、アプリ側で愛犬のために必要な栄養が取れるレシピを提案します。

## 機能候補

### MVP（最小限の機能）

- [ ] 愛犬の情報登録機能（年齢、体型、運動量、アレルギー）
- [ ] 選択した情報に基づいたレシピ検索・フィルタリング機能
- [ ] レシピ一覧表示機能(3~5件のレシピを表示)
- [ ] レシピ詳細表示機能

### 本リリース時に実装予定の機能

- [ ]レシピ提案数の拡張（１０件のレシピを提案）
- [ ]選択した犬種に応じて、適切なg数のみを表示
- [ ] ユーザー登録・ログイン機能
- [ ] 愛犬の情報保存機能（ログイン後に情報を保存できる）
- [ ] お気に入り機能
- [ ] SNS シェア機能（X、Instagram）
- [ ] レシピ投稿機能（管理者承認制）
- [ ] レシピへのコメント・レビュー機能

- [ ] 画像アップロード機能
